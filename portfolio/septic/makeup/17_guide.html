Для размещения фотографии на всю ширину контента следует применить класс img-fill

Для размещения нескольких фотографий предусмотрено несколько вариантов - в две, три и четыре колонки. Для стилизации только фотографий, нужно положить их в блок с классами layout и layout--flex и каждой фотографии применить класс img-half (для размешения в две колонки), img-third (в три колонки) или img-quarter (в четыре колонки)
Если под фотографиями нужно разместить описание, то следует применить такую же конструкцию, с одним отличие - классы img-half, img-third или img-quarter нужно применить к элементу div к этому же диву применить класс img-desc, а внутри div разместить фото c классом img-fill и описание

Также описание фотографии можно разместить не под фотографией, а непосредственно на ней. В этом случае описание будет видно при наведении. Для этого нужно фото задать свойством background для элемента с классом description__photo. Количество колонок регулируют аналогичные классы - description__photo--quarter и description__photo--third. Описание нужно положить внутрь, с классом description__photo-text

Опционально можно кастомизировать фотографии - для тени вокруг фото нужно применить класс shadow, для затемнения фото при наведении нужно разместить внутрь элемент с классом description__photo-hover

Списки
Для стилизации списков применять классы не требуются. Используйте списки UL и OL

Список UL

  • декорирование природными материалами;
  • декорирование искусственными объемными предметами по типу камня или элементами декора;
  • декорирование с помощью флористики;
  • декорирование с помощью отдельных конструкций.

Список OL

  1. декорирование природными материалами;
  2. декорирование искусственными объемными предметами по типу камня или элементами декора;
  3. декорирование с помощью флористики;
  4. декорирование с помощью отдельных конструкций.
