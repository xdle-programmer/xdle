Разработка приложений для Android

Платформа Google Android стремительно набирает популярность, и по данным на начало 2011 года устройств на базе этой операционной системе во всем мире продается больше, чем устройств на базе Apple iOS.

Преимущества платформы Android (по сравнению с Apple iOS и App Store):

  • большая открытость, меньше ограничений при разработке и публикации приложений;
  • большое разнообразие устройств, в т.ч. низкого ценового диапазона.

На что стоит обратить внимание при заказе мобильного приложения для Android

  1. Так как существует множество устройств на базе Android с разными разрешениями экрана, следует предусмотреть, чтобы интерфейс приложения смотрелся оптимально для всех вариантов.
  2. Для мобильного приложения для Android не существует понятия «стандартного дизайна», дизайн каждого приложения разрабатывается отдельно.
  3. Приложение может быть опубликовано в Google Market, либо распространяться отдельно, например, через ваш сайт.

Из чего складывается стоимость создания приложения для Android?

Стоимость приложения напрямую зависит от сложности его функционала и дизайна и складывается из количества часов, затрачиваемых на каждый этап разработки. Для точной оценки временных затрат на разработку создается техническое задание, которое включает в себя визуальный прототип мобильного приложения.

Составляющие стоимости приложения для Android:

Этап работ
Оценка временных затрат
Техническое задание На основе краткого описания задачи (брифа на разработку)
Дизайн На основе количества разрабатываемых эскизов пользовательского интерфейса, указанных в техническом задании
Программирование На основе перечня функций приложения и технических рисков, указанных в техническом задании.
Тестирование 5-10% от затрат времени этапа программирования
Публикация приложения в Android Market 1-2 часа

В рамках каждого этапа разработки обычно резервируется 5-10% времени на доработки по замечаниям и предложениям.
