Пример использования селекта

Подключение

Подключение максимально простое - нужно создать классический селект и после создать инстанс класса. Класс принимает три параметра:

Переключение на нативный селект задается в scss, в медиазапросе

События на селект транслируются в обе стороны, т.е. взаимодействовать с инстансом не нужно - для изменения значения можно изменить первоначальный селект и стриггерить событие 'change' на нем

            
selectClass:'select',     // String. Класс селекта, который требуется стилизовать (применимо, если селект на странице один)
$select:document.getElementsByClassName(this.selectClass)[0],       // node element, Селекта, который требуется стилизовать
customSelectClass:'custom-select',     // String. Класс, который получит кастомный селект
