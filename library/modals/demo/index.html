Использование

Для подключения нужно создать инстанс класса Modals

Открыть модалку по клику можно нужно добавив к кнопке дата атрибут data-modal-open с id модального окна

Для открытия модального окна через js, нужно вызвать метод .open('id модального окна')

            
window.modals = new Modals();

window.modals.open('test-modal')
