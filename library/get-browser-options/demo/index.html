Получение данных вьюпорта

https://www.npmjs.com/package/get-viewport-options

npm i get-viewport-options

Пример получения данных

Для подключения нужно создать инстанс класса GetViewportOptions

Получить ширину вьюпорта .getViewportWidth()

Получить высоту вьюпорта .getViewportHeight()

Получить доступность ховера .getHoverAvailability()

Получить ширину скролла .getScrollWidth()

Получить все опции .getFullOptions()

            
window.viewportOptions = new GetViewportOptions();

viewportOptions.getViewportWidth();

viewportOptions.getViewportHeight();

viewportOptions.getHoverAvailability();

viewportOptions.getScrollWidth();

viewportOptions.getFullOptions();
