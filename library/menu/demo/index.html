Использование

Для подключения нужно создать инстанс класса Menu и задать атрибуты кнопкам и меню

Атрибуты кнопок data-menu-target="###" и data-button-active-class="###"

Атрибуты меню data-menu-id="###", и data-menu-active-class="###"

Необязательные атрибуты меню, для переключения фона под меню data-background-selector="###", и data-background-active-class="###"

            
window.menu = new Menu();

Html структура

            
<div
                    data-menu-target="menu"
                    data-button-active-class="menu__button--open"
                    class="menu__button">
                Открыть меню с фоном
            </div>
            <div
                    data-menu-id="menu"
                    data-menu-active-class="menu__list--open"
                    data-background-selector=".menu__background"
                    data-background-active-class="menu__background--open"
                    class="menu__list">
                Содержимое меню
            </div>
